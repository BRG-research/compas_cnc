********************************************************************************
Installation
********************************************************************************

Stable
======

Stable releases are available on PyPI and can be installed with pip.

.. code-block:: bash

    pip install compas_cnc


Latest
======

The latest version can be installed from local source.

.. code-block:: bash

    git clone https://github.com/BRG-research/compas_cnc.git
    cd compas_tna
    pip install -e .


Development
===========

To install `compas_cnc` for development, install from local source with the "dev" requirements.

.. code-block:: bash

    git clone https://github.com/BRG-research/compas_cnc.git
    cd compas_tna
    pip install -e ".[dev]"
